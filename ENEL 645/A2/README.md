## Setup 🛠️

### Jupyter Setup

1. **Install Jupyter Notebook**
2. **Clone Our Repository**: `git clone <repo link>`
3. **Navigate to Our Project**: `cd notebooks`
4. **Launch Jupyter Notebook**: `jupyter notebook`

Ensure you have at least Python 3.10.0 installed.
